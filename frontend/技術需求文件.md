# 線上商店系統 - 技術需求文件

## 專案概述

### 專案名稱
線上商店系統 (Online Store Demo)

### 專案描述
一個功能完整的線上商店系統，包含客戶端購物介面和管理端後台系統，支援商品管理、訂單處理、使用者認證等核心電商功能。

### 版本資訊
- 版本: 1.0.0
- 開發日期: 2025年8月
- 授權: MIT

## 技術架構

### 前端技術棧
- **HTML5**: 網頁結構
- **CSS3**: 樣式設計與響應式佈局
- **Vanilla JavaScript (ES6+)**: 前端邏輯處理
- **Fetch API**: HTTP 請求處理
- **LocalStorage**: 客戶端資料儲存

### 後端技術棧
- **Node.js**: 伺服器運行環境
- **Express.js**: Web 框架
- **JWT (JSON Web Tokens)**: 使用者認證
- **bcrypt## 版本控制與部署

## API 設計規範

### 認證相關 API

#### POST /api/auth/register
- **功能**: 使用者註冊
- **請求格式**:
```json
{
  "username": "string",
  "email": "string",
  "password": "string"
}
```
- **回應格式**:
```json
{
  "success": true,
  "message": "註冊成功",
  "user": {
    "id": "string",
    "username": "string",
    "email": "string"
  }
}
```

#### POST /api/auth/login
- **功能**: 使用者登入
- **請求格式**:
```json
{
  "username": "string",
  "password": "string"
}
```
- **回應格式**:
```json
{
  "success": true,
  "token": "jwt_token",
  "user": {
    "id": "string",
    "username": "string",
    "email": "string"
  }
}
```

### 商品相關 API

#### GET /api/products
- **功能**: 取得商品清單
- **查詢參數**:
  - `category`: 商品分類
  - `featured`: 是否為精選商品
- **回應格式**:
```json
{
  "success": true,
  "products": [
    {
      "id": "string",
      "name": "string",
      "description": "string",
      "price": "number",
      "image": "string",
      "category": "string",
      "stock": "number",
      "featured": "boolean"
    }
  ]
}
```

#### POST /api/admin/products (需管理員權限)
- **功能**: 新增商品
- **請求格式**:
```json
{
  "name": "string",
  "description": "string",
  "price": "number",
  "image": "string",
  "category": "string",
  "stock": "number",
  "featured": "boolean"
}
```

### 訂單相關 API

#### POST /api/orders
- **功能**: 建立訂單
- **請求格式**:
```json
{
  "items": [
    {
      "productId": "string",
      "quantity": "number"
    }
  ],
  "shippingAddress": {
    "name": "string",
    "address": "string",
    "city": "string",
    "phone": "string"
  }
}
```

#### GET /api/orders (需認證)
- **功能**: 取得使用者訂單
- **回應格式**:
```json
{
  "success": true,
  "orders": [
    {
      "id": "string",
      "userId": "string",
      "items": "array",
      "total": "number",
      "status": "string",
      "createdAt": "string"
    }
  ]
}
```
